# Controlled AI for Regulated Environments — One-Pager

**Для:** decision makers (MRO, CAMO, OEM, QA)

---

## Проблема

Подготовка актов, техкарт и писем в авиационной/регулируемой среде — ручной поиск по документам, копирование, риск ошибок. Аудитор спрашивает: *«Откуда взято?»* — ответить сложно.

---

## Решение

**Papa App AI Document Workflow** — контролируемая генерация документов с доказуемой цепочкой источник → результат.

| Возможность | Ценность |
|-------------|----------|
| **Semantic search** | Поиск по смыслу (OpenAI / Ollama), не по ключевым словам |
| **EvidenceMap** | Кликабельная карта источников: фрагмент → оригинал → download |
| **Confidence score** | Релевантность каждого источника (0–100%) — прозрачно для аудитора |
| **Auto-suggest** | Автозаполнение полей из документов (S/N, inspector, act number) |
| **Draft → Confirm → Final** | Экспорт DOCX только после подтверждения. SHA-256 целостности. |

---

## Для кого

- **MRO / CAMO** — акты входного/выходного контроля, техкарты
- **OEM-подрядчики** — соответствие EASA / ARMAC
- **QA / Compliance** — прозрачность для регулятора

---

## Пилот (10–15 мин демо)

1. Поиск → confidence → auto-suggest → confirm → DOCX → sha256  
2. 1 изделие, 1 тип документа, 1 регулятор  
3. Success: время ↓, ручные правки ↓, прозрачность ↑
