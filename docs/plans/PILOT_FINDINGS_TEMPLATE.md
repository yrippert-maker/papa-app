## Известные ограничения (записи до пилота)

**OOM при полноразмерной индексации:** `docs:index:agent` падает по памяти. Mitigation: seed (2 docs) для демо. Отдельная задача — оптимизация индексатора (батчи, лимиты, streaming).

**Non-JSON на malformed input:** Agent search endpoint может вернуть non-JSON при em-dash/спецсимволах. Добавлена нормализация запроса (— → -) и guarded JSON fallback (200 + results: []).

**Prisma raw API (исправлено):** UI search при DATABASE_URL падал с «Invalid prisma.$queryRawUnsafe() invocation» — Prisma принимает только один аргумент (строку). В PgAdapter параметры теперь подставляются в одну строку с экранированием (`substituteParams` + `escapePgValue`), вызовы — `$queryRawUnsafe(sql)`, `$executeRawUnsafe(sql)` без доп. аргументов.
