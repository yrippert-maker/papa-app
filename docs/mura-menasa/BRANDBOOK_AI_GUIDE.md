# Руководство для AI-агента: стиль документов Mura Menasa

## Источник

Брендбук: `MURA MENASA FZCO brandbook.pdf`  
Путь к ассетам: `MURA MENASA/основа/MURA MENASA FZCO brandbook/`

## Цвета

- **Основной (красный):** `#EF1C23` (RGB 239, 28, 35)
- Цвет используется для акцентов, логотипа, кнопок, границ

## Логотип

- Файл: `/mura-menasa-logo.png` (в public)
- Использовать на фирменных бланках, заголовках документов

## Шрифты

- Основной: Inter, system-ui, sans-serif
- Для документов: допустимы Arial, Times New Roman при генерации DOCX

## Правила оформления документов

При генерации документов для Mura Menasa соблюдать:

1. **Руководство по качеству (РК)** — требования к оформлению служебных документов
2. **Руководство по процедурам ТОиР (MOPM)** — форматы актов, технологических карт
3. **АП-145** — авиационные правила МАК
4. **EASA** — European Union Aviation Safety Agency
5. **ICAO** — International Civil Aviation Organization

## Шаблоны

- **Фирменный бланк:** `templates/mura-menasa-firm-blank.docx`
- Использовать как основу для писем, актов, приказов

## Эталонные шаблоны (v2.0)

Все шаблоны хранятся в `templates/docx/brandbook/` и утверждены как эталоны.

| Файл | Тип | Intent API |
|------|-----|-----------|
| 01_Фирменный_бланк_MM.docx | Письма, приказы | mura-menasa-firm-blank |
| 02_АВК_шаблон_MM.docx | Акт входного контроля | act |
| 03_АВыхК_шаблон_MM.docx | Акт выходного контроля | act-output |
| 04_Техкарта_шаблон_MM.docx | Технологическая карта | techcard |

Стиль: логотип в шапке, красный акцент #EF1C23, Arial, футер с контактами и отметкой АП-145/EASA/ICAO.

## Конфигурация

Файл `config/mura-menasa-brandbook.json` содержит настройки для программного использования.
