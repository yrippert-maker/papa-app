# Эталоны Mura Menasa

Эталонные документы для ТВ3-117, АИ-9, НР-3 на основе ИКАО, EASA, АРМАК и внутренних руководств Mura Menasa.

## Структура

```
etolony/
  TV3-117/
  AI-9/
  NR-3/
  00_Общие_для_всех/
```

## Генерация через агент

**Источники:** `Новая папка/` и `Новая папка/menasa/` (образцы актов, руководств, ТК).

1. Индексировать: `DOCS_ROOT_DIR="/path/to/Новая папка" npm run docs:index:pgvector`
2. Dashboard → Помощник по документам
3. Поиск → выбор документов → Подготовить черновик → Подтвердить → Скачать DOCX
4. Сохранить в соответствующую папку

Подробно: [ETOLONY_RUNBOOK.md](./ETOLONY_RUNBOOK.md)
