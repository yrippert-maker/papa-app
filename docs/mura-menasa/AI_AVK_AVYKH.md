# AI для актов АВК/АВыхК

## Описание

Функция распознавания полей из фото актов входного/выходного контроля в окне «Помощник по документам».

## Как использовать

1. Выберите тип документа **«Акт»** или **«Бланк MM»**.
2. Нажмите **«Фото акта»** и загрузите фото (PNG, JPG).
3. AI (OpenAI Vision) извлечёт поля: S/N, P/N, инспектор, дата и т.д.
4. Нажмите **«Подготовить черновик»** — поля будут предзаполнены.
5. Доработайте черновик и экспортируйте в DOCX.

## Требования

- `OPENAI_API_KEY` — для работы Vision API (gpt-4o).
- Без ключа кнопка работает, но поля не распознаются (placeholder).

## API

- `POST /api/agent/extract-from-image` — FormData с `file` (image).
- Ответ: `{ extracted: {...}, source: "openai-vision" | "placeholder" }`.

## Извлекаемые поля

| Поле | Описание |
|------|----------|
| serial_number | Серийный номер (S/N) |
| part_number | P/N |
| act_number | Номер акта |
| inspector | ФИО проверяющего |
| approver | ФИО утверждающего |
| act_date | Дата акта |
| work_order | Заказ-наряд |
| product | Изделие |
| completeness | Полнота |
| condition | Состояние |
| decision | Решение |
